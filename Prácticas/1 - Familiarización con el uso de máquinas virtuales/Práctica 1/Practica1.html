Práctica 1: Instalación de Windows XP y Linux

1. Objetivo

En los dos ejercicios que componen esta práctica se verá el procedimiento de instalación, en sendas máquinas virtuales, de un Windows XP y de un Linux (concretamente una distribución Debian 7). Las razones para elegir estos sistemas operativos son las siguientes:

  1. Aunque Microsoft ha retirado ha totalmente el soporte a Windows XP, sigue existiendo aún mucho software en producción que corre estáblemente sobre XP (y en los casos en que no existan riesgos de seguridad es previsible que se mantendrá en producción aún durante bastante tiempo). Dado que el hardware más reciente ya no podrá ser soportado por el XP, por falta de drivers, la elección lógica es ejecutar el XP y software adicional sobre una máquina virtual, corriendo sobre un hipervisor en ese hardware moderno.

  2. Respecto a Debian, es una de las distribuciones no comerciales Linux más usadas, y forma la base de múltiples distribuciones comerciales (como, por ejemplo, Ubuntu).
