Práctica 6: Instalación y configuración de servidores de comercio electrónico

1. Objetivo

En esta práctica pondremos en uso los conceptos y procedimientos adquiridos en las prácticas anteriores, usando máquinas virtuales para instalar y poner en funcionamiento dos tiendas virtuales de comercio electrónico, usando para ello el sistema operativo Debian 7 con Apache2 y MySQL, sobre los que se instalarán los paquetes open-source de comercio electrónico TomatoCart (uno de los múltiples forks de osCommerce 3) y el propio osCommerce 2.3.4.

Adicionalmente veremos cómo usar dos servidores virtuales para distribuir en dos tiers físicos estas aplicaciones de tres capas, dedicando un servidor a la base de datos MySQL y dejando el otro servidor para las capas de Web y de lógica de negocio, y cómo generar pruebas de carga contra los servidores usando la herramienta Jmeter.

La práctica consta de los siguientes ejercicios:

Ejercicio 1 - Instalación de TomatoCart sobre un único servidor.

Ejercicio 2 - Instalación de osCommerce 2 sobre ese mismo servidor.

Ejercicio 3 - Reorganización en dos tiers, migrando la base de datos MySQL a un segundo servidor.

Ejercicio 4 - Elaboración de un Test Plan sencillo para Jmeter

Ejercicio 5 - Creación de cuentas Tomatocart usando Jmeter
